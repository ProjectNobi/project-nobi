# Contributing to Project Nobi

Thank you for your interest in contributing to Project Nobi! 🤖

## How to Contribute

1. **Fork** the repository
2. **Create** a feature branch (`git checkout -b feature/your-feature`)
3. **Commit** your changes with clear commit messages
4. **Push** to your fork
5. **Open** a Pull Request against `main`

## Code of Conduct

- Be respectful and constructive
- Focus on the mission: accessible AI companionship for everyone
- No discrimination, harassment, or toxic behavior

## Security Vulnerabilities

If you discover a security vulnerability, please report it responsibly:

- **Email:** security@projectnobi.ai
- **Do NOT** open a public GitHub issue for security vulnerabilities
- We will acknowledge receipt within 48 hours
- We will provide a timeline for a fix within 5 business days
